RegQueryValue

基本概念

RegQueryValue是Windows注册表API中的核心函数，用于读取注册表中指定键（Key）的默认值（未命名值） 数据。它是注册表读取操作的基础接口，需结合Windows注册表的结构（根键、子键、值项）使用。

返回值
错误码
ERROR_SUCCESS 读取成功。
ERROR_FILE_NOT_FOUND 指定的子键或默认值不存在。
ERROR_MORE_DATA 缓冲区大小不足，lpcbValue会输出所需的最小缓冲区大小。
ERROR_ACCESS_DENIED 权限不足（例如打开 HKEY_LOCAL_MACHINE 时未以管理员身份运行）。
ERROR_INVALID_HANDLE hKey是无效的注册表键句柄。

RegQueryValueEx

基本概念

RegQueryValueEx是Windows注册表API中功能更强大的读取函数，相比RegQueryValue，它支持读取注册表键的「命名值项」（如Path、Version）和所有注册表数据类型（REG_SZ、REG_DWORD、REG_BINARY等），是实际开发中读取注册表的首选接口。
