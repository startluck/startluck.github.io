注册表类API-RegEnumValue

发表于2025-11-21|更新于2025-11-22
|浏览量:

基本概念

RegEnumValue是Windows注册表API 中的值项枚举函数，用于遍历指定注册表键（HKEY）下的所有「值项」（包括默认值），获取每个值项的名称、数据类型、数据大小等信息。配合循环调用，可完整枚举键下所有值项，是注册表批量读取、备份等场景的核心接口。

基本语法

返回值
错误码
ERROR_SUCCESS（0） 枚举成功，获取到当前索引的值项信息。
ERROR_NO_MORE_ITEMS（259） 所有值项已枚举完毕，无更多数据。（循环终止条件）
ERROR_MORE_DATA（234） 缓冲区不足（名称或数据缓冲区），lpcchValueName/lpcbData 输出所需大小。
ERROR_ACCESS_DENIED（5） 权限不足（hKey 无 KEY_ENUMERATE_SUB_KEYS 或 KEY_READ 权限）。
ERROR_INVALID_HANDLE（6） hKey是无效句柄。
ERROR_INVALID_PARAMETER（87） 参数无效（如lpReserved非NULL）。

具体示例

示例枚举HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion下的所有值项，输出每个值项的「索引、名称、类型、数据大小」，并读取REG_SZ类型的值项数据
